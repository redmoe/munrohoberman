Puzzle game about trying to get bots into folders, sokoban-esque. Your bot sends out signals which tells other bots to move the same way. The problem? Delayed response time! They get the message a turn later so you have to plan your moves carefully.

I had a lot of fun drawing old tech since the game has an 80's computer aesthetic. I was going to use them as a reward for completing levels. Try to name all the devices if you can!
